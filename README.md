# Blakley Plumbing Inc. — Website

Premium, responsive marketing website for **Blakley Plumbing Inc.**, a family-owned
plumbing company serving Fort Wayne, Indiana since **1958**.

- **Live:** open `index.html` in any browser (zero build step).
- **Stack:** single self-contained HTML file + embedded CSS/JS, custom SVG icons,
  scroll-reveal + parallax, animated "line monitor" water animation, Google Maps embed.
